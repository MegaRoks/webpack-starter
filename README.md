# Webpack Starter

A simple set, for a quick start.

#Technologies used in the project

- Babel
 - Webpack
 - Scss
 - Css
 - Styles
 - PostCSS
 - JS
 - Autoprefixer
 - jQuery

 # Getting Started


| commands        | actions                |
|-----------------|------------------------|
|`npm install`    | install                | 
|`npm run dev`    | development            |  
|`npm run build`  | production             |
